# Profile Image

프로필 사진을 이 폴더에 저장하세요.

## 파일 이름
- `profile.jpg` 또는 `profile.png`로 저장하세요
- 현재 설정: `profile.jpg`

## 권장 사항
- **크기**: 400x400 픽셀 이상 (정사각형 권장)
- **형식**: JPG 또는 PNG
- **파일 크기**: 200KB 이하 권장

## 사용 방법
1. 프로필 사진 파일을 이 폴더(`assets/images/`)에 저장
2. 파일 이름을 `profile.jpg` 또는 `profile.png`로 변경
3. `_config.yml`의 `author.avatar` 경로 확인
4. Jekyll 서버 재시작

## 현재 설정
`_config.yml`에서 다음 경로로 설정되어 있습니다:
```yaml
avatar: "/assets/images/profile.jpg"
```
